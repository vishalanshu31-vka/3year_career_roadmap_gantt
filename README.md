# 🗺️ 3-Year CS Career Roadmap — Gantt Chart

> **"From zero to product-company ready in 1095 days."**
> A visual, interactive Gantt chart for Computer Science students aiming to become Software Engineers at top product-based companies (Google, Amazon, Microsoft-level).

---

## 📌 What Is This?

This is a **day-by-day interactive Gantt chart** that maps out a complete 3-year learning roadmap for a CS student targeting a software engineering role. It covers 10 skill tracks across 1095 days, broken into 10–20 day learning blocks with specific topics, practice goals, and deliverable outputs for every phase.

Built as a single-file HTML application — no installation needed. Just open in any browser.

---

## 🚀 How to Use

1. Download `3year_career_roadmap_gantt.html`
2. Open the file in any modern browser (Chrome, Firefox, Edge)
3. Click the **Year tabs** at the top to switch between phases
4. **Hover over any colored block** to see:
   - Topics to study
   - Practice goals (number of problems / projects)
   - Output / deliverable (GitHub repo, project, certificate)

## 💡 Tips for Following This Roadmap

- **Consistency beats intensity** — 3 hours daily is better than 10 hours on weekends
- **Build, don't just watch** — every topic needs a mini-project or practice problem
- **Push to GitHub daily** — even small commits build your green grid
- **Join communities** — Discord servers, LinkedIn, college coding clubs
- **Track progress weekly** — use a Notion page or Google Sheet
- **Revise every 30 days** — revision cycles are built into the chart
- **Don't skip DSA** — it is the single most important factor in getting hired
